# Glossary

- API: A stable boundary where one component requests behavior or data from another.
- ABI: Binary compatibility boundary; critical in C and C++ libraries.
- RAII: Resource Acquisition Is Initialization; bind resource lifetime to C++ object lifetime.
- RAG: Retrieval-Augmented Generation; combine retrieval from trusted documents with model generation.
- Contract: Versioned schema for messages, APIs, or function boundaries.
- State machine: A model with explicit states and transitions.
- Sandbox: Restricted execution environment for untrusted or configurable code.
- Ownership: The component responsible for releasing or mutating a resource.
- Idempotent: Safe to repeat without changing the result after the first successful execution.
- Least privilege: Grant only the minimal permissions needed.
