## Verification baseline

This knowledge base was created against the following primary references checked on 2026-06-13:

- Python: official Python 3.14.6 documentation, including the language reference, library reference, tutorial, `venv`, `asyncio`, `sqlite3`, `logging`, `unittest`, and packaging guides: <https://docs.python.org/3/>.
- C: ISO C current published revision is C23 / ISO/IEC 9899:2024; production examples here deliberately use portable C17 unless a C23 feature is explicitly named. Public reference: <https://en.cppreference.com/w/c>.
- Lua: official Lua manuals. Lua 5.5 is listed as the current manual by lua.org; Lua 5.4 remains the conservative production baseline for many embedders. References: <https://www.lua.org/manual/> and <https://www.lua.org/manual/5.4/>.
- C++: current published ISO C++ standard is C++23 / ISO/IEC 14882:2024; examples use C++20-compatible idioms unless a C++23 feature is called out. Reference: <https://isocpp.org/std/the-standard> and <https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines>.
- Flask: official documentation: <https://flask.palletsprojects.com/>.
- pytest: official documentation: <https://docs.pytest.org/>.
- CMake: official documentation: <https://cmake.org/documentation/>.
- OWASP: application security guidance: <https://owasp.org/www-project-top-ten/> and <https://cheatsheetseries.owasp.org/>.

Facts are separated from recommendations where a project decision can vary.
